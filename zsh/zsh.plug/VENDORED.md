# Vendored from Oh My Zsh

These files are copied verbatim from [Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh)
(MIT License) so the zsh config no longer needs the full framework installed:

- `plugins/vi-mode/vi-mode.plugin.zsh`
- `plugins/golang/golang.plugin.zsh`
- `plugins/rust/rust.plugin.zsh`
- `lib/git.zsh` — git prompt helpers (used by the 1995parham fallback theme)
- `lib/async_prompt.zsh` — async backend for the git prompt helpers

Source commit: ohmyzsh/ohmyzsh@ff2f16e8

To refresh, re-copy these paths from a current Oh My Zsh checkout.
